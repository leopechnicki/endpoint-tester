# Changelog

All notable changes to this project are tracked here. Release notes going
forward are generated by
[release-please](https://github.com/googleapis/release-please-action) from
[Conventional Commits](https://www.conventionalcommits.org/).

## [0.2.0] - 2026-04 (historical)

- refactor: move `SupportedFormat` to `types.ts` as single source of truth
- fix: validate `--framework` flag against known frameworks before scanning
- fix: create parent directory for `scan --output` path
- fix: validate `--format` flag in generate command
- fix: deduplicate endpoints by method+path in `Scanner.scan()`
- fix: escape control characters in string literals and validate baseUrl
- feat: v0.3 — response validation, smart body, query params, enhanced
  auth, 3 new adapters (Fastify, Koa, NestJS)
- feat: framework auto-detection, improved adapter robustness, expanded
  tests
